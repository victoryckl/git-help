    [编辑] 
    1.查看本地分支：git branch

    [kelvin@localhost
    syva_project]$ git branch
    board_ema3530
    * master
    NOTE：*代表当前分支

    [编辑] 2.查看远程分支：git
    branch -r

    [kelvin@localhost
    syva_project]$ git branch -r
    origin/board_ema3530
    origin/master
    NOTE:说明远程有2个分支：一个是master分支，一个是board_ema3530分支

    [编辑]
    3.如何在本地创建一个分支:git
    branch test //test是分支名

    [kelvin@localhost
    syva_project]$ git branch test
    [kelvin@localhost
    syva_project]$ git branch
    board_ema3530
    * master
    test
    NOTE：可以看到，我们在本地生成一个新的分支，名字叫test，但是远程服务器上并没有此分支，我们使用git
    branch -r查看
    [kelvin@localhost
    syva_project]$ git branch -r
    origin/board_ema3530
    origin/master

    [编辑]
    4.如何切换到其他分支：git
    checkout test

    [kelvin@localhost
    syva_project]$ git checkout
    test
    Switched to branch 'test'
    [kelvin@localhost
    syva_project]$ git branch
    board_ema3530
    master
    * test
    NOTE:
    （1）如果项目比较大，切换分支是需要等待一段时间的；
    （2）切换分支成功后，会提示“Switched
    to branch 'test'”；
    （3）*代表当前分支，说明我们已经处于test分支了。
    [编辑]
    5.如何将本地新建立的分支提交到远程服务器:git
    push origin test:test

    [kelvin@localhost
    syva_project]$ git push origin
    test:test
    Total 0 (delta 0), reused 0
    (delta 0)
    To
    git@192.168.1.11:syva_project.git
    * [new branch] test -> test
    ------------------------------------------------------------
    我们再使用git branch
    -r查看一下远程分支，看是否提交成功：
    [kelvin@localhost
    syva_project]$ git branch -r
    origin/board_ema3530
    origin/master
    origin/test
    OK，可以看到，远程服务器上已经生成了新的分支test。
    接下来，我们就可以在test分支上进行我们自己的开发了。

    [编辑] 7.如何合并分支：git
    merge

    [编辑]
    （1）我们先切换到master分支看一下

    [kelvin@localhost
    syva_project]$ git checkout
    master
    Switched to branch 'master'
    [kelvin@localhost
    syva_project]$ git branch
    board_ema3530
    * master
    test
    [kelvin@localhost
    syva_project]$ vim README
    date :2010/11/04
    version:1.0.2
    1.I have intergrated mp3dec
    into DVSDK.
    并没有发现我们新添加的2行：
    +date :2010/11/17
    +1.Add new branch test
    [编辑] （2）合并分支

    [kelvin@localhost
    syva_project]$ git merge test
    Updating 38bb715..fc6c0ff

    1.Add new branch test
    说明合并已经成功。

    [编辑] 8.如何删除分支

    [编辑] （1）如何删除远程分支

    [kelvin@localhost
    syva_project]$ git push origin
    :test//远程的test将被删除，但是本地还会保存的
    To
    git@192.168.1.11:syva_project.git
    - [deleted] test
    [kelvin@localhost
    syva_project]$ git branch -r
    origin/board_ema3530
    origin/master
    [kelvin@localhost
    syva_project]$ 
    OK，现在我们已经删除了远程分支test，但是本地并没有删除。

    [编辑] （2）如何删除本地分支

    [kelvin@localhost
    syva_project]$ git branch
    board_ema3530
    * master
    test

    [kelvin@localhost
    syva_project]$ git branch -D
    test//强制删除本地分支test
    Deleted branch test (was
    fc6c0ff).
    [kelvin@localhost
    syva_project]$ git branch
    board_ema3530
    * master
    [kelvin@localhost
    syva_project]$ 
    OK，现在本地的test分支也已经被删除了。
